## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites
To run this game, you need to have the * [Rust](https://www.rust-lang.org/tools/install) programming language.

### Installation & Run

1. **Clone the repository**

   ```bash
   git clone https://github.com/yatsenk/silent-night.git
   cd silent-night
   
2. **Build and Run**

   ```bash
   cargo run
